FC++ library structure

For most applications, the easiest way to use the library is just to 
   #include "prelude.h"
but here is the header dependency diagram, in case you are interested. 
                       config
                        /  |
                 signature |
<functional>,      |       |
<utility>        curry     |
        \        /         |
         operator     ref_count
               \      /
               function
                 |
               reuse  <iterator>,<new>,<cstdlib>,<exception>
                 |    /
                 |   /
                 list
                  |
                prelude
