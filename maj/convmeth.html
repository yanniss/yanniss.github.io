Convenience Methods

There are time you might want to call methods defined for the AST classes that represent the code that you've just created. One most oftenly used is unparse(). You can call unparse on any meta-variable, and the result is a java.lang.String that is the code you've quoted. This is handy when you want to print the final piece of code you've created into a file.
